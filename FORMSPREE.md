# Formspree Integration

The landing4 waitlist form uses **Formspree** to handle submissions.

## Current Setup

**Formspree Endpoint**: `https://formspree.io/f/xjgeblaz`

This endpoint receives all form submissions with:
- Email
- Phone
- Role (Buyer/Seller/Both)

## Benefits

✅ **No backend required** - Formspree handles everything  
✅ **No API keys** - Works out of the box  
✅ **Email notifications** - You get notified of new submissions  
✅ **Submission dashboard** - View all submissions at formspree.io  
✅ **Spam protection** - Built-in honeypot and reCAPTCHA  

## Viewing Submissions

1. Go to [formspree.io](https://formspree.io/)
2. Log in with your account
3. View the "xjgeblaz" form submissions

## Export/Integration

Formspree supports:
- CSV export
- Webhook notifications
- Zapier integration
- Google Sheets auto-sync (paid plan)
- Email forwarding

## Testing

Form is working! Submissions appear in your Formspree dashboard immediately.

## No Environment Variables Needed

Unlike Google Sheets, Formspree requires **zero configuration**. The endpoint is public and safe to use in client-side code.
